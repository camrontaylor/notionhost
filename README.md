# NotionHost

Turn Notion pages into fast, SEO-optimized documentation sites. No API setup required.

## Getting started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000). Copy `.env.example` to `.env.local` and set `DATABASE_URL` and `NEXTAUTH_SECRET` for auth.

See the project PRD in the repo for full specs and build phases.
